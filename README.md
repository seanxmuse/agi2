# 20back - Healthcare Visit Management System

A React-based healthcare application for managing patient visits, generating clinical documentation, insurance billing, and patient engagement artifacts.

## Features

- **Dashboard** - Overview of daily schedule and patient queue
- **Visit Detail** - Comprehensive visit management with 3 artifacts
- **Insurance Notes** - 3-layer documentation with ICD-10 codes
- **Clinical Orders** - Prescriptions, labs, imaging, and referrals
- **Patient Portal** - Personalized patient-facing summaries

## Tech Stack

- React 18 + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- React Router v7
- Radix UI primitives

## Getting Started

```bash
npm install
npm run dev
```
